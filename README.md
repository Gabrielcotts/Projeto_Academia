# Sistema de Gerenciamento de Academia em Java

Este projeto consiste em um sistema de gerenciamento de academia desenvolvido em **Java**, utilizando os princípios da **Programação Orientada a Objetos (POO)**. O objetivo é simular o gerenciamento de alunos e avaliações físicas, colocando em prática conceitos fundamentais da linguagem Java e boas práticas de desenvolvimento de software.

O sistema funciona por meio de um menu interativo no console, permitindo o cadastro de alunos, cálculo do IMC, gerenciamento de avaliações físicas e consulta de informações, proporcionando uma aplicação prática dos conceitos estudados durante a formação em Engenharia de Software.

## Funcionalidades

* Cadastro de alunos;
* Validação do nome do aluno;
* Verificação de maioridade;
* Cálculo do Índice de Massa Corporal (IMC);
* Aplicação de desconto na mensalidade;
* Escolha de planos da academia (Mensal, Trimestral, Semestral e Anual);
* Cadastro de avaliações físicas;
* Cálculo da maior, menor e média das avaliações;
* Validação das notas das avaliações;
* Exibição de relatórios com os dados do aluno e das avaliações;
* Menu interativo em ambiente de console.

## Conceitos aplicados

* Programação Orientada a Objetos (POO);
* Encapsulamento;
* Construtores sobrecarregados;
* Métodos de validação;
* Atributos estáticos (`static`);
* Arrays;
* Estruturas de decisão (`if`, `switch`);
* Estruturas de repetição (`for`);
* Modularização do código;
* Organização em pacotes;
* Entrada de dados utilizando `Scanner`.

## Tecnologias utilizadas

* Java
* JDK 25
* IntelliJ IDEA

## Objetivo do projeto

Este projeto foi desenvolvido como parte da minha jornada de aprendizado em Java, com o objetivo de consolidar conceitos de Programação Orientada a Objetos por meio da construção de uma aplicação prática. Durante o desenvolvimento, foram aplicados princípios de modelagem de classes, encapsulamento, validação de dados e organização do código, preparando a base para projetos mais robustos utilizando tecnologias como JavaFX, Spring Boot e bancos de dados relacionais.
